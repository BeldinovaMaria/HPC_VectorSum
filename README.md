## 📌 Задание

Реализовать сложение элементов вектора двумя способами:
1. **На CPU** — последовательное суммирование
2. **На GPU** — параллельная редукция с использованием CUDA

Размер вектора: от 1000 до 1 000 000 элементов.

---

## ⚙️ Реализация

### CPU (`sum_vector_cpu`)
- Простой последовательный цикл
- Сложность O(n)

### GPU (`sum_kernel` + `sum_vector_gpu`)
- **Редукция** — каждый поток суммирует часть вектора
- Размер блока: **256 потоков**
- Использование **shared memory** для ускорения
- После редукции внутри блока — суммирование результатов блоков на CPU

### Что распараллелено?
- Вектор разбивается на блоки
- Каждый поток независимо суммирует свою часть
- Редукция внутри блока — параллельное попарное сложение
- **Почему**: сумма элементов — независимая операция, идеально подходит для GPU
